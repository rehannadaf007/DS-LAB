# 🧠 Data Structures Lab Programs (Python)

A collection of **Data Structures programs implemented in Python** to understand fundamental concepts like arrays, linked lists, stacks, queues, and sorting algorithms.

---

## 📌 About

This repository contains practical implementations of core **Data Structures (DS)** concepts using Python.  
It is designed to strengthen problem-solving skills and provide hands-on experience with commonly used data structures.

---

## 📚 Programs Included

### 🔹 Arrays (Lists in Python)
- Traversal  
- Insertion & Deletion  
- Searching (Linear, Binary)

### 🔹 Linked List
- Singly Linked List  
- Insertion & Deletion  
- Traversal  

### 🔹 Stack
- Implementation using List  
- Push & Pop operations  

### 🔹 Queue
- Simple Queue  
- Circular Queue  

### 🔹 Sorting Algorithms
- Bubble Sort  
- Selection Sort  
- Insertion Sort  

### 🔹 Searching Algorithms
- Linear Search  
- Binary Search
